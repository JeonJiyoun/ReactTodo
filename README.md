## 📺 프로젝트 실행 화면

## ✔️ 주요 기능
### 1. Todo 추가
### 2. Todo 삭제
### 3. Todo 편집

## ⚒️ 기술 스택
- React.js
- Typescript
- Redux
- Webpack & Babel

## 📖 실행 방법
```
git clone https://github.com/JeonJiyoun/ReactTodo.git
cd src
npm install
npm run start
```
